Hamlet, by William Shakespeare

The ARIA Tree control is keyboard accessible. 

Set focus to the ARIA Tree.
Press the Up and Down Arrow keys to navigate vertically.
Press the Left and Right Arrow keys to collapse or expand branch nodes.
Press any letter or number to jump to a tree node starting with that character or digit.
Press Enter or the Spacebar to activate a node.
Mouse users can click tree branches and nodes to expand/collapse or activate items as usual.
