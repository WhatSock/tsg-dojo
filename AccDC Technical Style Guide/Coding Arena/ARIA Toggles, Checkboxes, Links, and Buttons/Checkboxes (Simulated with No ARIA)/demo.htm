CHECKBOXES: What do you want to be when you grow up?
Standard Image Link (A tag with an embedded IMG) as a toggle:
A Horse! A Horse!
Standard A tag with a background image as a toggle:
A Donkey! A Donkey!

The checkbox controls are keyboard accessible. 

Press the Tab key to navigate between the Checkboxes. 
Press the Enter key to toggle the state of the selected checkbox.
Mouse users can click a checkbox to toggle the state as usual.
